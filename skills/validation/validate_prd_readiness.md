# Skill: validate_prd_readiness

## Purpose
Check that a PRD is complete enough to break into stories.

## Inputs
- prd_markdown

## Reads
- context/standards/writing.md

## Checks
- Problem, Users, Goals present
- Metrics present (even if TBD)
- Risks + assumptions captured
- Open questions listed

## Outputs
- readiness_result (pass/fail + reasons)
- Suggested fixes
